# منصة اختبارات تحديد المستوى - دليل التشغيل المحلي

تم تحديث هذه النسخة وإصلاحها لتعمل مع قاعدة بيانات **PostgreSQL** على منصة **Render**، مع تفعيل نظام الاختبارات التفاعلي والذكاء الاصطناعي.

## المتطلبات الأساسية

- **Node.js** (إصدار 18 أو أحدث)
- **pnpm** (مثبت عبر `npm install -g pnpm`)
- **قاعدة بيانات PostgreSQL** (يمكنك استخدام الرابط الخارجي المزود من Render)

## خطوات التشغيل المحلي

### 1. تثبيت التبعيات

افتح مجلد المشروع في محرر الأكواد (VS Code) وقم بتنفيذ الأمر التالي:

```bash
pnpm install
```

### 2. إعداد متغيرات البيئة

قم بإنشاء ملف باسم `.env` في المجلد الرئيسي للمشروع وأضف الإعدادات التالية (استخدم الرابط الخارجي لقاعدة البيانات):

```env
DATABASE_URL=postgresql://testexamwep_user:5K3CsgQXV5EWxceW3gOuI7vpGtR5kuFO@dpg-d6pior2a214c73c14vg0-a.singapore-postgres.render.com/testexamwep
NODE_ENV=development
PORT=3000
```

### 3. مزامنة قاعدة البيانات (اختياري)

إذا كانت قاعدة البيانات فارغة وتريد إنشاء الجداول، قم بتشغيل:

```bash
pnpm db:push
```

### 4. تشغيل المشروع

لتشغيل الخادم وواجهة المستخدم في وضع التطوير:

```bash
pnpm dev
```

سيكون الموقع متاحاً على الرابط: `http://localhost:3000`

## الإصلاحات التي تمت

1.  **قاعدة البيانات:** تحويل النظام بالكامل من MySQL إلى PostgreSQL ليتوافق مع Render.
2.  **واجهة الاختبارات:** تفعيل جلب الأسئلة الحقيقية، نظام التصحيح التلقائي، وحساب الدرجات لكل قسم.
3.  **الذكاء الاصطناعي:** تفعيل زر "اسأل الذكاء الاصطناعي" داخل الاختبار لشرح الأسئلة الصعبة.
4.  **صفحة النتائج:** ربطها بالبيانات الحقيقية لعرض تقييم شامل وتوصيات بناءً على الدرجة.
5.  **الأمان:** تحديث منطق التحقق من الأكواد للاختبارات المدفوعة.

## ملاحظة هامة

تأكد من استخدام **External Database URL** عند التشغيل من جهازك الخاص، بينما يستخدم الموقع على Render الـ **Internal Database URL**.
