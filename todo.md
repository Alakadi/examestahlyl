### المتطلبات الحالية:
- Node.js 18+
- PostgreSQL
- pnpm

### خطوات التشغيل:
```bash
pnpm install
pnpm db:push
pnpm dev
```

### نقاط الدخول:
- الصفحة الرئيسية: `http://localhost:3000`
- تسجيل دخول Admin: `http://localhost:3000/api/dev/login`
- لوحة التحكم: `http://localhost:3000/admin`
